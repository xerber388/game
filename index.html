Как разместить эту игру бесплатно?

1. GitHub Pages

Создайте аккаунт на GitHub, создайте репозиторий, загрузите этот HTML-файл как index.html и активируйте GitHub Pages в настройках.

2. Netlify

Зарегистрируйтесь на Netlify и перетащите папку с файлом игры в область загрузки. Получите бесплатный домен.

3. Vercel

Создайте аккаунт на Vercel, загрузите файл через интерфейс или подключите GitHub для автоматического развертывания.
